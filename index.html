Get started with Trek

Use npm to install from the command line, then check out the Quick Start Guide. Read the documentation.

# Install the `trek-cli` command globally
npm install trek-cli -g

# Create a trek app and chdir
trek new app && cd app

# Install as a development dependency
npm install

# Start trek app
npm start
